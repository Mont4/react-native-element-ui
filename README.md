# react-native-element-ui

## Getting started

`$ yarn add react-native-element-ui`

## Usage
```javascript
import {Text, H1, H2, H3, H4, H5, H6} from 'react-native-element-ui';

<Text bold={true}>Hi</Text>
<H1 bold={true}>Hi</H1>
<H2 bold={true}>Hi</H2>
<H3 bold={true}>Hi</H3>
<H4 bold={true}>Hi</H4>
<H5 bold={true}>Hi</H5>
<H6 bold={true}>Hi</H6>
```
